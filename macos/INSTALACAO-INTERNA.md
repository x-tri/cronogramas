# Instalação interna — XTRI Cronogramas macOS

1. Extraia `XTRI-Cronogramas-macOS-interno.zip`.
2. Arraste `XTRI Cronogramas.app` para `/Applications`.
3. Se o macOS bloquear, clique com botão direito no app e escolha `Abrir`.
4. Se ainda bloquear, libere em `Ajustes do Sistema > Privacidade e Segurança`.

Esta distribuição é interna, assinada ad-hoc e sem notarização pública Apple.
